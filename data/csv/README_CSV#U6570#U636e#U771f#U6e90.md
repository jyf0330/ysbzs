# CSV 数据真源说明

这个目录是新核心默认读取的数据源。以后改数据优先改这里的 CSV。

## 读取规则

- 程序启动 / 新建局时优先读取 `data/csv/*.csv`。
- 如果 `data/csv` 不存在，才回退到 `data/normalized_data.json`。
- 改 CSV 后，运行 `npm run check:all` 验证；如果 UI 服务已经启动，点“新建局”或重启 `npm run ui` 让新数据重新载入。

## 主要文件

- `01_宠物主表_好读版.csv`：宠物基础属性、元素、机制、形状文字。
- `02_怪物模板_好读版.csv`：敌方模板属性。
- `03_怪物波次_好读版.csv`：第几天、第几回合刷什么敌人。
- `04_机制词条库_60条.csv`：机制 ID 和机制说明。
- `05_事件主表_好读版.csv`：商店事件、事件收益、代价。
- `06_商店奖励池_好读版.csv`：商店池、奖励池、价格、权重、解锁日。
- `07_遗物祝福_好读版.csv`：遗物/祝福奖励。
- `08_形状行动槽_好读版.csv`：宠物行动槽、元素、形状。
- `09_跨表校验.csv`：人工校验说明。
- `10_初始阵容.csv`：我方初始宠物和站位；想换开局阵容改这里。

## 注意

- 不要改列名，程序按列名读取。
- 多个机制用英文逗号、中文逗号或顿号分隔都可以，例如 `mech_aura,mech_curse`。
- 多个标签/池 ID 可以用 `,`、`，`、`、` 分隔。
- `行(1-8)` 和 `列(1-8)` 是给人看的 1 起始坐标，程序会转成内部 0 起始坐标。
