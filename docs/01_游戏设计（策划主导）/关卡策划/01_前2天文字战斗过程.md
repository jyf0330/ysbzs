## 基础设定

### 我方初始单位

**火苗灵** HP 20，攻击形状 SD[1] 右1格，行动槽 火/火/火
定位：近战，清前排怪物

**滴滴灵** HP 20，攻击形状：跳过第1格、命中第2、3格（新远程形状）
行动槽 水/水/水
定位：远程，越过前排打城堡

### 敌方

**敌方城堡** HP 100，棋盘右侧
每回合召唤怪物，怪物挡在城堡前方。
最终目标。

### 伤害说明

元素独立叠层。同元素多槽命中同一目标：
第1槽 = 1层 = 1伤害
第2槽 = 2层 = 1+2 = 3伤害
第3槽 = 3层 = 1+2+3 = 6伤害

⚠ 四元素独立叠层。火+水+火 = 火2,水1，不是火3。

## 第1天 上午

### 第1回合

**回合开始：** 敌方城堡召唤 小怪A（HP6, ATK1, AP5），挡在城堡前方。

**怪物行动预览：**
小怪A M1左移 M2左移 M3进入攻击范围 → 目标：火苗灵，预计 -1

**玩家预览：**

火苗灵（SD[1] 右1格）→ 小怪A。3火槽命中 → 小怪A 火3层 = 6伤害。HP 6→0。

滴滴灵（远程跳过首格）→ 第1格空，第2格小怪A（预期死亡），第3格敌方城堡。3水槽命中城堡 → 水3层 = 6伤害。

总预览：小怪A HP 6→0，敌方城堡 -6。

**执行：**
火苗灵向右打出火焰，小怪A连续被命中。
滴滴灵释放远程水弹，越过空格直击敌方城堡。

**结算：**
小怪A 火3层 = 6伤害，HP 6→0，被击败。
敌方城堡 水3层 = 6伤害，HP -6。
怪物阶段：无存活怪物，不行动。

**提示：** 你击败了第一只怪物。滴滴灵的远程攻击也打到了城堡。城堡才是最终目标。

## 第1天 下午

### 第1回合

**回合开始：** 敌方城堡召唤 小怪C（HP4, ATK1）、小怪D（HP4, ATK1）。两只怪物分散站位在城堡前方上侧和下侧。

**系统提示：** 敌人开始分散站位。只靠向右攻击不能同时处理两个方向。试试刚买的十字使。

**怪物行动预览：**
小怪C：M1左移 M2左移 M3 → 火苗灵（-1）
小怪D：M1左移 M2左移 M3 → 火苗灵（-1）

**玩家预览：**

十字使十字攻击（SD[12]）同时覆盖小怪C和小怪D：
- 小怪C 风3层 = 6伤害，HP 4→0
- 小怪D 风3层 = 6伤害，HP 4→0

火苗灵继续向右打城堡：火3层 = 6伤害。

滴滴灵没有好角度，保留行动。

总预览：小怪C HP 4→0，小怪D HP 4→0，城堡 -6。

**执行：**
十字使释放十字风刃，同时命中上方和下方的怪物。
火苗灵向右攻击，火焰命中敌方城堡。

**结算：**
小怪C 风3层 = 6伤害，HP 4→0，被击败。
小怪D 风3层 = 6伤害，HP 4→0，被击败。
敌方城堡 -6。
怪物阶段：无存活怪物。

**提示：** 你用十字攻击一次清掉2只怪物。购买新英雄可以改变你的攻击方式。

### 第2回合

**回合开始：** 敌方城堡召唤 冲锋怪（HP8, ATK2, AP3），比普通小怪更靠近我方。

**怪物行动预览：**
冲锋怪 M1左移 M2进入攻击范围 M3攻击火苗灵（-2）
目标：火苗灵，预计伤害 -2

火苗灵卡片显示：⚠ 冲锋怪即将攻击，预计 -2

**系统提示：** 怪物不是木桩。没被击败的怪物会移动、接近、攻击。看到 ⚠ 时，要判断先清怪还是继续打城堡。

**玩家预览——两个选择：**

选择A（全力清怪）：
- 火苗灵 → 冲锋怪：火3层 = 6伤害
- 十字使 → 冲锋怪：风3层 = 6伤害
- 冲锋怪 HP 8→0，火苗灵不受伤，城堡不受伤

选择B（贪城堡伤害）：
- 滴滴灵 → 城堡：水3层 = 6伤害
- 火苗灵 → 冲锋怪：火3层 = 6伤害
- 冲锋怪 HP 8→2（未死！），冲锋怪攻击火苗灵（-2），城堡 -6

玩家选择A。

**执行：**
火苗灵攻击冲锋怪，冲锋怪火元素叠到3层。
十字使补十字风刃，风元素叠到3层。

**结算：**
冲锋怪 火3层（6伤害）+ 风3层（6伤害）= 12伤害
HP 8→0，被击败。
怪物阶段：冲锋怪已死亡，不行动。

**下午结束：**
当前队伍：火苗灵 HP20，滴滴灵 HP20，十字使 HP18。
进入晚上商店。

## 第2天总结

玩家已学会：
1. 不同英雄有不同攻击形状（近战/远程/十字）
2. 商店卖英雄，同名英雄可以合成（青铜+青铜→白银）
3. 合成到白银后伤害大幅提高
4. 怪物会移动和攻击，没死的怪物会反击
5. 威胁预览帮助判断先清怪还是继续打城堡
6. 城堡是长期目标，要逐步推进
