### 伤害速查（教学锚点）

| 叠层 | 累计伤害 | 典型来源 |
|---|---|---|
| 同格 3 层 | **6** | Lv1 三槽各 +1 层（火苗灵等同格集火） |
| Lv2 合成 | **以预览为准** | 槽形状变化，不写死 6 层/21 |

- 公式：`n×(n+1)/2`；写伤害时必须 **层数 + 累计伤害 + HP 前后** 一致。
- **集火**：多英雄同一目标按行动顺序逐步扣 HP。

# 第3天：精英关

## 第3天 上午

城堡当日回满：100。

### 第1回合（教学：集火）

**回合开始：** 敌方城堡召唤 **铁甲队长**（HP24, ATK3, AP3，精英）挡在城堡前方，两侧召唤 小怪A（HP4, ATK1）、小怪B（HP4, ATK1）。小怪B 站位较远。

**系统提示：** 出现精英怪。铁甲队长血厚（HP24），单个英雄一回合打不死。需要**集火**——让多个英雄攻击同一目标。

**怪物行动预览：**
铁甲队长：M1左移 M2左移 M3 → 火苗灵，预计 −3（火苗灵卡显示 ⚠ −3）
小怪A：M1左移 M2左移 M3 → 滴滴灵，预计 −1
小怪B：距离远，本回合打不到我方。

**玩家预览：**

火苗灵（白银）→ 铁甲队长：火6层 = 21 伤害，HP 24→3（未死）。
滴滴灵（白银）→ 铁甲队长：火6层 = 21 伤害，HP 3→0（被击败）。
十字使（青铜）→ 小怪A：火3层 = 6 伤害，HP 4→0。

总预览：铁甲队长 HP 24→3→0（集火两步），小怪A HP 4→0，小怪B 无威胁，我方不受伤。

**执行：**
火苗灵打出强化火焰，铁甲队长被削到残血。
滴滴灵远程火弹补刀，铁甲队长倒下。
十字使十字火焰清掉侧边小怪A。

**结算：**
铁甲队长：火苗灵 火6层 = 21，HP 24→3；滴滴灵 火6层 = 21，HP 3→0（补刀，伤害溢出仍按 6 层结算）。
小怪A：十字使 火3层 = 6，HP 4→0，被击败。
怪物阶段：小怪B 距离远，不行动。

**提示：** 你用两个英雄集火，一回合打掉了血厚的精英。单打不死的目标，就让多个英雄打同一个。

### 第2回合（教学：取舍）

**回合开始：** 敌方城堡再召唤 铁甲队长2（HP24, ATK3, AP3），并在上、下两侧各召唤 小怪C（HP4, ATK1）、小怪D（HP4, ATK1），分散站位。

**系统提示：** 这一回合你做不到「全清且零损失」。要学会**取舍**：先保证大威胁，再决定能不能兼顾小威胁。

**怪物行动预览：**
铁甲队长2：M1左移 M2左移 M3 → 火苗灵，预计 −3（⚠ −3）
小怪C：M1左移 M2左移 M3 → 滴滴灵，预计 −1
小怪D：M1左移 M2左移 M3 → 十字使，预计 −1

**玩家预览——两个选择：**

选择A（集火精英，吃小怪伤害）：
- 火苗灵 → 队长2：火6层 = 21，HP 24→3
- 滴滴灵 → 队长2：火6层 = 21，HP 3→0（击败）
- 十字使 → 小怪C+小怪D：十字同时覆盖，各 **火3层 = 6**，HP 4→0
- 结果：全清，但十字使若只能覆盖一侧……需要判断站位

选择B（先保英雄不掉血）：
- 火苗灵 → 队长2：火6层 = 21，HP 24→3（未死）
- 十字使 → 小怪C+小怪D：清掉两只
- 滴滴灵 → 城堡或保留
- 结果：队长2 残血 3，下回合更靠近、仍会攻击火苗灵 −3

玩家选择A：十字使站位调好，十字范围同时覆盖小怪C、小怪D。

**执行 & 结算：**
火苗灵 + 滴滴灵 集火击败队长2。
十字使十字火焰同时清掉小怪C、小怪D。
怪物阶段：全灭，不行动，我方零损失。

**提示：** 集火解决精英、范围解决小怪群。看懂威胁预览，就能判断这一回合该先打谁。

**上午结束：** 进入中午商店。

**系统提示：** 你现在有 5 类英雄：前排 / 远程 / 范围 / 召唤 / 辅助。这一天学**怎么搭配**——让常规输出和召唤引擎同时运转。

### 第1回合（教学：引擎 + 常规输出并行）

**回合开始：** 城堡召唤 铁甲队长（HP24, ATK3）+ 小怪群 3 只（各 HP4）。

**怪物行动预览：**
铁甲队长：M3 → 火苗灵（⚠ −3）
小怪群：分别 → 滴滴灵 / 十字使 / 召芽灵（各 −1）

**玩家预览（一套构筑动作）：**

- 火苗灵 → 铁甲队长：火6层 = 21，HP 24→3。
- 滴滴灵 → 铁甲队长：火6层 = 21，HP 3→0（集火补刀）。
- 十字使 → 小怪E / F / G：各 **火6层 = 21**，HP 4→0。
- 召芽灵 **召唤** 召唤物T（绒语灵被动：HP8、ATK2 起步）。
- 泉泉灵 **治疗** 召唤物T：HP 拉满，**ATK +1 → 3**。
- 召唤物S（Day4 留存，ATK2）+ 召唤物T：**本回合无相邻残怪**，自动攻击不触发。

引擎统计预览（本 run 累计）：召唤 2 次 / 治疗 2 次。

**执行 & 结算：**
集火秒精英、范围清小怪群、引擎再添一只强化召唤物并治疗成长。
`runSummonActions`：场上召唤物（S、T）攻击相邻残怪（本回合已清空，无目标则待命）。
怪物阶段：全灭，我方零损失。

**提示：** 同一回合里，常规英雄解决眼前威胁，召唤引擎在后台越滚越大。这就是「构筑组合」。

### 第2回合（教学：引擎滚雪球 + 死亡留火连锁雏形）

**回合开始：** 城堡召唤 围城群——盾怪 2 只（各 HP10）+ 小怪 4 只（各 HP4），数量压上来。

**系统提示：** 怪多了，正面输出忙不过来。让召唤物去**前线消耗**——它们会自动攻击、被打死后留火，火叠起来能引爆清群。

**玩家预览：**

- 火苗灵 → 盾怪甲：火6层 = 21，HP 10→0。
- 十字使 → 盾怪乙 + 小怪（十字覆盖）：盾怪乙 **火6层 = 21**，HP 10→0；覆盖到的小怪 **火6层 = 21**，HP 4→0。
- 滴滴灵 → 城堡：火6层 = 21，HP 100→79（远程线畅通时）。
- 召芽灵 **再召唤** 召唤物U（绒语被动强化），顶在怪群前。
- 泉泉灵 **治疗** 召唤物 S/T：各 **ATK +1**（S：2→3，T：3→4）。
- 召唤物 **自动攻击**补刀：若有残血邻格怪，按当前 ATK 扣 flat 伤害（不走叠层公式）。

引擎统计预览（本 run 累计）：召唤 3 次 / 治疗 4 次（本回合治疗 S、T 各 1 次）。

**执行 & 结算：**
前排/远程/范围三线把盾怪和小怪清空；召唤引擎补刀并继续扩编。
若后续怪潮中召唤物被打死：原地留火，多只死同格 → 火叠 3 层 → 结算十字引爆，额外清怪（连锁雏形，正式多元素克制第6天以后教）。
怪物阶段：残怪被召唤物补掉，我方零损失。

**提示：** 引擎成型后，召唤物自己就能清掉一部分怪。你越往后越省力——这是召唤流的「滚雪球」。

**上午结束：** 进入中午商店。

## 第5天下午（半自由）

下午起进入「半自由战斗」：城堡持续召唤混合怪潮，玩家用已成型的阵容自行决策。重点不再是单条教学，而是**让玩家自己运转一整套构筑**：

## 第5天总结

玩家已学会：
1. 五类英雄（前排 / 远程 / 范围 / 召唤 / 辅助）如何在**同一回合并行**运转。

4. 被动英雄（绒语灵 / 爆爆灵）默默强化整套引擎。
