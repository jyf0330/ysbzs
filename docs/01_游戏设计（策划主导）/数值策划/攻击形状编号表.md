# 一、形状编号表 v0.1

## 1. 编号规则

商品名统一用：

```txt
元素·几格·几号·几阶
```

例如：

```txt
火·5格·25号·1阶
水·3格·4号·2阶
土·6格·30号·1阶
```

含义：

```txt
火 = 元素
5格 = 这个攻击块由5个格子组成
25号 = 形状编号
1阶 = 当前阶级
```

## 2. 形状文本规则

```txt
x = 占用格
. = 空格
```

程序规则：

```txt
所有形状默认朝右。
左上角为坐标原点。
后续通过方向块旋转到上、下、左、右。
```
