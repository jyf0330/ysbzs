## 游戏概述

- 游戏名称：元素背包史（ysbzs）
- 类型 / 题材：回合制棋盘战术 / 元素策略 Roguelite
- 目标平台：浏览器（单 HTML 文件）
- 目标玩家：轻度策略向桌游爱好者

## 核心体验

- 一句话体验目标：通过在背包中排列元素方块、控制两位英雄的走位，在 13×13 棋盘上把怪物逐波消灭。
- 玩家主要感受：计划性强、每回合有多个决策点、元素连击爆发感。

## 核心玩法

- 核心操作：选中英雄移动 + 选中行动槽使用攻击方块，怪物自动执行 3AP 回合。
- 核心循环：玩家回合（移动 + 攻击）→ 怪物回合（3AP 移动/攻击）→ 商店（购买 / 合成方块）→ 下一波。
- 胜利条件：消灭所有波次的怪物（每波 5 个小回合）。
- 失败条件：所有英雄 HP 归零。

## 一局流程

1. 开始：两位英雄和怪物出现在棋盘两端，玩家获得初始行动槽。
2. 中段推进：每波 5 个小回合，玩家通过元素攻击和走位击败怪物；波结束后进入商店购买/合成升级行动槽。
3. 结算：当前版本无明确结算界面；通关条件 `[NEEDS_USER_INPUT]`。
