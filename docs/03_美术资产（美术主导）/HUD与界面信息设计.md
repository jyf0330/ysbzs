## 三、右侧信息面板（#rp）

### 3.1 回合信息 box

| DOM ID | 内容 | 样式 |
|---|---|---|
| `#ph` | 当前回合阶段文字 | 20px 加粗，金色 `#f0a500`，居中 |
| `#rc` | 第N波 · 第X/5小回合 | 11px，灰色 `#888`，居中 |
| `#gi` | 💰 N 金币 | 12px，右对齐 |
| `#exa` | ⚡ 一键执行 按钮 | 蓝色 `#1565C0`，与 `#etb` 同风格，13px |
| `#etb` | 结束回合 → 按钮 | 蓝色 `#1565C0`，100%宽，13px |

**回合阶段对应文字：**

| phase | 显示文字 |
|---|---|
| PLAYER | ⚔️ 玩家回合 |
| MONSTER | 👾 怪物行动中… |
| SHOP | 🛒 商店 |
| OVER | 💀 游戏结束 |

### 3.2 英雄状态 box（#hs）

每位英雄一行（`.hi` flex 行）：

- 左侧：英雄名（英雄A蓝色 `#1565C0` / 英雄B绿色 `#2E7D32`，13px 加粗）+ 坐标（10px 灰色）
- 右侧：HP 数值（12px）+ HP 条（宽 90px，高 5px，圆角 3px）

**HP 条颜色规则：**

| HP 占比 | HP 条颜色 |
|---|---|
| > 50% | 绿色 `#22c55e` |
| 25%~50% | 黄色 `#eab308` |
| < 25% | 红色 `#ef4444` |

### 3.3 行动点 box（#asl）

行动槽以 **2 列 `.asl-grid`** 排列，每个迷你卡（`.as-mini`）紧凑显示：

| 区域 | 内容 |
|---|---|
| `.asm-head` | 圆形编号徽章（元素色背景）+ 英雄标签（A-1/B-1…）+ 已使用✓ |
| `.asm-name` | 元素名·槽号·×倍率（如 火·5号·×2），元素色显示 |
| `.asm-dir-row` | 4个方向按钮横排 ↑ ↓ ← →，每个 20×16px |
| 底部 flex 行 | 英雄选择下拉框（flex:1）+ ⚔/✓ 使用按钮（auto宽） |

**行动槽状态样式：**

| 状态 | 类名 | 视觉 |
|---|---|---|
| 普通 | `.as-mini` | `#0d1f3c`，1px `#1e3a5f` |
| 选中 | `.as-mini.sel` | `#1a2f50`，`#f0a500` 描边 |
| 已使用 | `.as-mini.used` | opacity 0.45 |

**方向按钮（`.asd`）：**

| 状态 | 视觉 |
|---|---|
| 普通 | `#0d1f3c`，1px `#1e3a5f`，灰色文字 |
| 激活 | `#f0a500` 背景，黑色文字，`#f0c040` 描边 |

### 3.4 战斗日志 box（#log）

- 固定高度 110px，overflow-y:auto，10px，行高 1.6
- 文字颜色：`#9ca3af`（灰白）
- 每条日志下方有 1px `#1a1a1a` 分割线
- 最新条目在底部（scroll 到底）

## 四、覆盖层 / 模态层

### 4.1 商店（#so）

- 触发时机：波次结算后，玩家点"结束回合"进入商店
- 样式：fixed 全屏遮罩，背景 rgba(0,0,0,0.9)，z-index:100
- 内容容器 `#sc`：max-width 680px，居中，金色 2px 描边，圆角 12px

| 区域 | 说明 |
|---|---|
| `.stt` | 🛒 商店 标题，22px 金色居中 |
| `.gd` | 💰 金币数，右对齐，金色加粗 |
| `#scat` | 分类商品列表（攻击方块 / 道具 / 元素强化 / 背包操作） |
| `.csb` | ✅ 完成购买按钮，绿色 `#15803d`，100%宽 |

**商品条目（`.si`）：**

- flex 横排：名称 + 形状预览 | 价格（金色） | 购买按钮
- 已购按钮变蓝（`.bb.sl`），钱不够变灰

### 4.2 元素 Tooltip（#tt）

- 触发：点击棋盘元素格
- 位置：fixed，跟随点击位置
- 内容：各元素叠层数 + willExplode 状态 + 引爆按钮（手动触发 settleExplosions）
- 样式：金色 2px 描边，`#161b22` 背景，z-index:150

### 4.3 消息弹窗（#msg）

- 触发：胜利/失败/提示类消息
- 位置：fixed 屏幕中央，transform:translate(-50%,-50%)
- 样式：金色描边，z-index:200
- 按钮：金色背景，点击关闭

### 4.4 行动点分配弹窗（#ap-modal）

- 触发：从背包装备行动方块时，需指定分配给哪个行动槽
- 位置：fixed 屏幕中央，z-index:300
- 内容：文字提示 + 动态生成槽按钮列表（`.apb`）+ 取消按钮
- 样式：蓝色 2px 描边，min-width 220px
