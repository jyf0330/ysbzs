# 美术风格指南 · 元素背包史

## 文档状态

- 状态：草稿（Demo v0.1 羊皮纸手绘风）
- 最后更新：2026-06-01

## 整体风格

- 风格定位：浅色羊皮纸 / 手绘线稿风，无图片资源，纯 CSS + Emoji 呈现。
- 色调：暖色纸面底色（`#f2eadc`），深棕色文字（`#4a3f35`），低饱和柔和配色。
- 氛围：童话战棋 / doodle tactics / 手账风，不要暗黑、不要金属、不要霓虹。
- 字体：优先日系圆体/手写体（Hiragino Maru Gothic ProN、Klee、YuKyokasho），回退到系统无衬线。

## 线条风格

- 边框统一使用细实线或虚线（1-2px），颜色为暖棕色系。
- 无 box-shadow 发光效果；选中状态用虚线描边替代光晕。
- 格子间隙用棋盘底色透出，模拟纸上画线感。
