# 美术风格指南 · 元素背包史

## 文档状态

- 状态：草稿（Demo v0.1 极简风格）
- 最后更新：2026-05-24

## 整体风格

- 风格定位：极简暗色调 UI，无图片资源，纯 CSS + Emoji 呈现。
- 色调：深色背景（`#0d1117`），配色以元素色为主调，强调色为金色（`#f0a500`）。
- 字体：系统默认 emoji + 等宽字体，无外部字体依赖。

## 色彩规范

| 用途 | 颜色值 |
|---|---|
| 背景 | `#0d1117` |
| 格子背景 | `#161b22` |
| 格子边框 | `#21262d` |
| 英雄 A（蓝） | `#1565C0` |
| 英雄 B（绿） | `#2E7D32` |

| 地元素 | `#d97706` |
| 怪物预警（攻击）| `#ef4444` 描边 |
| 怪物预警（移动）| `#f59e0b` 描边 |

## 单位卡片规范

- 英雄卡：38×38px 圆角矩形，上方显示字母（A/B），下方显示 ❤HP/maxHP。
- 怪物卡：38×38px 圆角矩形，上方显示 👾，下方显示 ❤HP ⚔ATK。
- 伤害徽章：居中顶部，透明背景+文字阴影，不遮挡 HP/ATK 数字。

## Emoji 用法

| Emoji | 用途 |
|---|---|
| 👾 | 怪物标识 |
| ❤ | HP |
| ⚔ | ATK |
| 🔥💧🌿⛰ | 元素引爆特效（日志）|
