## CSS 类名规范

| 类名前缀 | 用途 |
|---|---|
| `.hm` | 英雄标记（hero marker）|
| `.hma` | 英雄 A 专属样式 |
| `.hmb` | 英雄 B 专属样式 |
| `.hm-letter` | 英雄字母（A/B）|
| `.hm-hp` | 英雄 HP 文字 |
| `.mm` | 怪物标记（monster marker）|
| `.mm-icon` | 怪物图标（👾）|
| `.mm-stats` | 怪物 HP/ATK 文字 |
| `.em` | 元素方块标记（element marker）|
| `.cell` | 棋盘格子 |
| `.hp-ovl` | 英雄攻击预览叠层 |
| `.dmg-badge` | 预计伤害徽章（怪物格）|
| `.hero-warn-badge` | 预计受伤徽章（英雄格）|
| `.mw-atk` | 怪物攻击预警格 |
| `.mw-mov` | 怪物移动预警格 |
