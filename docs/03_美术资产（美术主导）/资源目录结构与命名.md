## CSS 类是视觉表现的唯一边界

View 层渲染函数（`renderBoard`、`renderHS`、`renderSlots`、`renderTurn`）只通过 CSS 类和内联 style 颜色值控制视觉。

视觉规则描述在 ViewModel（`buildBoardVM` 等）中以 `classes[]` 数组、`pvEl`、`pvOpacity`、`monDmg` 等字段输出，View 层直接应用，不重新计算。

因此：
- 若要修改元素格背景色 → 修改 `EB` / `EC` 常量（Core 层）
- 若要修改预警格显示方式 → 修改 CSS `.mw-atk` / `.mw-mov`
- 若要修改伤害徽章样式 → 修改 CSS `.dmg-badge` / `.hero-warn-badge`
- 若要修改预览叠层透明度逻辑 → 修改 `buildBoardVM()` 中的 `pvOpacity` 赋值
