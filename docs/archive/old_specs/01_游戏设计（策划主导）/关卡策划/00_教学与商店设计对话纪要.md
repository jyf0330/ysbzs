## 一、核心设计方向

### 1. 世界观方向：攻打敌方城堡

**已确认：**
- 敌方城堡是棋盘右侧的最终目标，HP 100（草案）
- 怪物由敌方城堡召唤，挡在城堡前方
- 玩家的最终目标：摧毁敌方城堡（预计第 10 天左右）
- 城堡是**长期压力源**
- 前 2 天不能打爆城堡

**原实现（需修改）：**
- 当前城堡 `pos=(12,1)` 在我方左下角，是怪物攻击的目标
- 胜负条件：城堡被毁→失败

### 2. 城堡血量节奏

| 时间 | 教学目的 | 城堡 HP 变化 |
|---|---|---|
| 第1天 | 教基础循环（近战、远程、清怪、打城堡） | 草案 |
| 第2天 | 英雄升级（同名合成升阶） | 前2天不能打爆 |
| 第3天 | 精英关（完整战斗判断） | 本日主要打精英，城堡掉血少 |
| 第4天 | 英雄技能（主动/被动） | 技能提升输出 |
| 第5天 | 构筑组合（半自由战斗） | 初步阵容搭配 |
| 第6-9天 | 元素克制、复杂组合 | 逐步推进 |
| 第10天 | 最终目标 | 预计摧毁城堡 |

> ⚠️ 每日扣血数值是草案，不能写死为正式数值。

### 3. 商店方向

**本轮（第一版）：**
- **只卖英雄**，不卖行动槽、不卖元素强化、不卖消耗品
- 同名两个英雄合成升一阶（核心商店成长）
- 价格按品级统一
- 找 The Bazaar（bazaar-engine）作为参考

**后续扩展（本轮不做）：**
- 多摊位标签商店
- 消耗品

## 二、品级与定价系统（草案）

### 品级方案

ywh 提出的方向：青铜 → 白银 → 钻石（参考 The Bazaar 的 Bronze/Silver/Gold/Diamond）

当前游戏已有的框架：1阶 → 2阶 → 3阶

用 1阶/2阶/3阶 或 Bronze/Silver/Gold 都可以，品级就是升级路线。

### 定价原则

```
所有同品级英雄价格一致。
不从形状格子数算价格，不从元素稀有度算价格。
价格 = 品级基准价。
```

### 英雄标签系统

每个英雄有 5-6 个标签（用于后续多摊位扩展）：

| 标签类别 | 示例值 |
|---|---|
| 元素 | 火、水、风、土 |
| 位置 | 前排、后排、中排 |
| 攻击类型 | 近战、远程、范围 |
| 种族 | 人族、兽族、元素灵 |
| 职责 | 输出、防御、辅助、召唤 |

### 示例

| 英雄 | 标签 |
|---|---|
| 火苗使 | `火, 前排, 近战, 人族, 输出` |
| 水滴使 | `水, 后排, 远程, 人族, 输出` |
| 十字使 | `风, 中排, 范围, 元素灵, 输出` |
| 岩盾使 | `土, 前排, 近战, 人族, 防御` |

## 三、核心战斗规则

### 元素叠层
四元素独立叠层。火+水+火 = 火2, 水1，不是火3。

### 怪物行为
- 怪物不是"只会向左攻击"
- 教学关可以简化，但核心仍是寻路+威胁预览
- 怪物会移动、接近英雄、攻击最近目标

### 格子交互
- 点击格子默认查看详情
- 不是点击手动引爆（除非后续专门设计手动引爆系统）

### 伤害公式
```
layers * (layers + 1) / 2
1层=1, 2层=1+2=3, 3层=1+2+3=6
```

## 四、前 5 天教学结构

### 第1天：基础战斗
- 教：近战攻击、远程攻击、清怪、打城堡、商店买英雄
- 流程：
  - 上午：**2 个小回合**
  - 中午：商店（只卖英雄）
  - 下午：分散站位的怪物，十字使发挥作用
- 不是完整的 5 小回合后才进商店

### 第2天：英雄升级
- 教：**同名两个英雄合成升一阶**
- 升级后英雄 HP 和攻击都变强
- 高血量怪需要升阶英雄处理
- 前2天不能打爆城堡

### 第3天：精英关
- 教：完整战斗判断——怪物移动、威胁预览、集火、取舍
- 精英怪铁甲队长（HP24, ATK3）+ 小怪群
- 本日主要打精英，城堡只掉少量血

**重点：** 只教"完整战斗判断"，不塞元素克制、遗物、复杂 Buff。元素是四元素独立叠层。

### 第4天：英雄技能
- 教：英雄有主动技能、被动技能
- 解锁英雄技能系统
- 召唤宠物只是技能示例之一，不要把"召唤使"写死成唯一方案

### 第5天：构筑组合
- 教：前排、远程、十字、召唤、辅助之间怎么搭配
- 玩家已有多种英雄类型，开始决定培养方向

### 第6天以后
- 元素克制、遗物、复杂 Buff 放到第 6 天以后
- 前三天先让玩家会玩，后面再让玩家玩得深
