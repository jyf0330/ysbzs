## 0. 与代码对齐约定（2026-05-30 决策）

| 项 | 约定 |
|---|---|
| 元素 | **与代码一致**：火苗灵=火、滴滴灵=水、十字使=风；召唤引擎=水+召唤、死亡**留水** |
| 合成 | 同英雄 **Lv1→Lv3**（三合一 `mergeUnits`），文档称 **「Lv2 合成」**；伤害按**实际槽形状叠层**，以游戏内预览为准 |
| 城堡 | **敌方**城堡 HP 100，**每新开一天回满**；**我方**城堡 HP **跨天保留** |

| 金币 | 初始 10 + **击杀掉落** + 每晚 `nightIncome` + **利息**（非阶段固定表） |
| 商店价 | 界面 **Tier1=3 金 / Tier2=6 金**（`SHOP_PRICE_CONFIG.unitPrice`）；另售消耗品 |

## 1. 相对站位词表

写文字过程时，优先用下表词汇，避免一上来写格子坐标。

| 词汇 | 含义（相对关系） | 典型刷怪/布局 | 服务哪条教学 |
|---|---|---|---|
| **挡在城堡前方** | 怪物在「城堡 ↔ 我方」主通路上 | 单只小怪、护城怪 | 教近战清前排、远程能否越过打城堡 |
| **分散站位（上侧 / 下侧）** | 至少两只怪不在同一行或同一列 | 小怪C + 小怪D | 教十字范围、买新英雄改打法 |
| **侧边站位** | 偏主路一侧 | 盾怪 + 小怪E | 教前排集火 + 侧翼范围 |
| **更靠近我方 / 冲锋怪** | AP 高、更快进入攻击范围 | 冲锋怪 HP8 AP3 | 教 ⚠ 威胁预览 |
| **距离远 / 本回合打不到** | 预览 M1–M3 仍进不了攻击范围 | 盾怪B、后排小怪 | 教优先级 |
| **双精英同屏** | 两只高血怪同时出现 | 双铁甲队长 | 教集火取舍 |
| **精英 + 小怪群** | 1 精英 + 多只杂兵 | 铁甲队长 + 小怪 | 教集火 + 范围 |
| **围城群 / 数量压上** | 多前排 + 多小怪 | 盾怪×2 + 小怪×4 | 教构筑成型 |
| **左侧 / 右侧空格** | 路径上的空位，可站人/召唤 | 护城怪左侧空格 | 教召芽灵召唤 |
| **路径被挡** | 目标格被召唤物占用 | 召唤物S 顶在怪前 | 教召唤物当肉盾 |
| **远程线畅通** | 滴滴灵跳过首格后命中城堡或后排 | 前排被清 | 教远程蹭城堡 |
| **十字覆盖区** | 十字使 SD[12] 罩住多只怪 | 分散站位 | 教范围清怪 |

## 2. 商店定价（tier 体系）

### 2.1 代码实际规则

- 商店英雄标价：**Tier1 = 3 金**，**Tier2 = 6 金**（`genShop` 使用 `SHOP_PRICE_CONFIG.unitPrice`）。
- `UNIT_DEFS.cost` / 品级基准价（青铜2 等）供设计参考；**文字过程购物句以 tier 价为准**。
- 另售消耗品（元素强化、回复药等），价格见 `genShop` 内 `conTypes`。

### 2.2 教学池出现节奏（`SHOP_POOLS`）

| 阶段 | 池 key | 典型新机制英雄 |
|---|---|---|
| 第1天 中午 | `day1_midday` | 十字使 |
| 第1天 晚上 | `day1_night` | 合成火苗/核心重复 |
| 第2天 | `day2_*` | 核心英雄 + 余烬灵等 |
| 第3天 中午 | `day3_midday` | 合成十字等 |
| 第3天 晚上 | `day3_night` | **召芽灵**（首次） |
| 第4天 中午 | `day4_midday` | **泉泉灵** |
| 第4天 晚上 | `day4_night` | **绒语灵**、爆爆灵 |

## 4. 伤害叙述（Lv1 / Lv2）

- 公式：`explDmg(n) = n×(n+1)/2`（与代码一致）。
- **Lv1 三槽同格叠 3 层火** → **6 伤害**（教学常用口径）。
- **Lv2 合成**后槽位 **形状变化**（如 `sn:2,2,3`），同格层数**不保证 6 层**；正文写「Lv2 强化」时，**以游戏预览数字为准**，不写死「21」。
