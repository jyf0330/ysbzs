# 元素背包史 V1 联动数据包（火水风 30宠）

生成时间：2026-06-08 07:10

## 这次和上一版的区别

这不是单独为第7天自建格式，而是在用户上传的 `csv.7z` 原有数据格式基础上改：保留 `00~10` 这些主 CSV 的命名和核心列，并新增 `11~15` 作为必要联动表。

## 当前拍板规则

- 第一版只做三元素：火、水、风。
- 最终宠物池：30个宠物。
- 元素比例：火12、水10、风8。
- 品质比例：青铜12、白银10、黄金8。
- 钻石不占30宠主池，用于进阶、试炼原型、终局。
- 效果分公式：`HP + 3 × (攻 + 防) + 5 × (盾 + 行动) + 机制分`。
- 盾每回合结束后重生到最大盾。
- 我方宠物移动不消耗行动力，但攻击后不能再移动。
- 敌方移动消耗行动力。
- 攻击行动块默认：先造成物理伤害，再附加元素层，再触发元素反应。

## 第7天火核心试炼

第7天默认阵容：

1. 融焰娘：黄金火核心。
2. 火绒狐：白银火启动。
3. 冲浪鸭：白银水催化。
4. 疾风隼：白银风聚火，承接原讨论里的“旋风狸/旋风收束”功能，避免新增旧表外宠物名。

敌方：兽群统领固定第1行第8列，复制体在右上4×4区域随机合法占位。测试固定为：精灵龙R1C6、皮皮鸡R2C7、骑士蜂R3C6、棉悠悠R4C5。

首回合验收：1黄金+3白银必须击杀骑士蜂黄金复制体和精灵龙黄金复制体；4黄金阵容首回合应能解决3个怪。
