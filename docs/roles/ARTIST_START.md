## 当前美术方向

- 浅米色羊皮纸底（`#f2eadc`）。
- 黑色/深棕手绘线稿感。
- 低饱和淡彩，暖棕配色体系。
- 可爱 Q 版小精灵/小怪兽（emoji 呈现，无外部图片依赖）。
- 棋盘是视觉第一焦点。
- 格子必须整齐、正方形。
- 血条、单位、文字不能出格。
- UI 不能高饱和、不能复杂页游感。
- 使用 CSS 渐变、双线边框、纸面内阴影实现质感，不引入外部图片。

## 生图/美术任务原则

- 先看指定参考图和素材索引。
- 不要重新发明风格。
- 不要随意换物种、换角色、换棋盘规格。
- 输出提示语时必须写清：画风、布局、棋盘、单位、按钮、禁忌项。
- 参考图路径见 `05_ASSET_AND_FILE_INDEX.md`。
