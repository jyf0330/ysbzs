## 当前策划铁律

- 商店第一版**只卖英雄**。
- 英雄同名 2 合 1 升阶，最高 lv3。
- 整体向 The Bazaar 结构对齐。
- 本项目节奏更快，高档内容更早出现。
- 第一阶段优先跑通火伤害和召唤流。
- 水治疗、风牵制、土阻挡等复杂特殊能力可以 pending。
- 战斗内无随机（零隐藏随机，零暴击/闪避）。
- 旧规则不自动继承，必须看当前文档。

## 策划任务原则

- 先判断是否符合当前阶段。
- 不要同时展开太多系统。
- 不要为了酷炫增加无法实现的机制。
- 每个方案要说明收益、风险、落地成本。
- 修改数值要同步更新 `03_CURRENT_NUMBERS.md`。
- 重大决策要记录到 `06_DECISION_LOG.md`。
