## 核心逻辑

| 规则 | 说明 |
|------|------|
| 商店 tier | 根据 `calcShopTier(day)`：d1-2→1, d3-4→2, d5-6→3, d7-10→4 |
| 商品生成 | `genShop()`：保留冻结商品，按格子数从同 tier 池中取，超出部分从 `UNIT_TIER_POOL` 补齐 |
| 购买 | 金币不足提示；购买后同名自动合成；英雄槽满（>2 active）放入备战 |
| 出售 | 返还 `unit.level` 金币（LV1=1, LV2=2, LV3=3） |
| 刷新 | 消耗 1 金币，清除冻结列表，重新 genShop |
| 冻结 | switch 模式：冻结/解冻同一商品，刷新时保留 |
| 合成 | 同名 2→1，最大 4 级，升级后赠送一次免费高级英雄 |
| 合成送英雄 | `addLevelupUnit()`：shopTier+1 的 tier 池中随机取一个免费加入商店 |
