## 怪物能力说明

| 能力 | 机制 | 来源代码 |
|------|------|----------|
| block_path | onMoveIntent·pending（占位，未实装完整逻辑） | MONSTER_TYPES.blocker |
| target_castle | onChooseTarget·pending（占位，未实装完整逻辑） | MONSTER_TYPES.siege |
| iron_wall | 受击减免：伤害>5时减2（onBeforeTakeDamage·pending） | MONSTER_TYPES.boss5 |
| guard_aura | onChooseTarget·pending（占位） | MONSTER_TYPES.minion |
| lava_surge | onRoundStart 触发，怪物所在格铺1层火 | battle.js:394-399 |
| core_split | 每n回合(onEveryNthRound)分裂1只普通怪到相邻空格 | battle.js:400-422 |

## 注意

- 标 `pending` 的能力表示 data.js 中已定义 status:'pending'，实际战斗逻辑未完整实现
- `lava_surge` 和 `core_split` 已经在 battle.js 的 `runMonsterAbilityHook` 中实现
- 怪物 HP/ATK 是基础值，波次生成时 `buildWaveForDay` 按原值生成，不缩放
