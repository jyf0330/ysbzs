## 波次生成逻辑

1. `buildWaveForDay(day, phase)` 读取配置
2. 在预算内按 `rngPick` 权重随机填充怪物，直到预算用完或达到 `maxAlive`
3. 如果该场战斗配置了强制 Boss，确保 Boss 在怪物列表中
4. 教学波跳过以上逻辑，直接硬编码
5. 出生点：`getSpawnCells(spawnSize)` 在棋盘右上角 `(0..spawnSize-1, 8-spawnSize..7)` 区域分配
