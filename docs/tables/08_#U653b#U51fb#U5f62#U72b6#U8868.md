## 方向旋转规则

方向通过 `rotCells(cells, dir)` 实现：

| 方向 | 变换 |
|------|------|
| right | 不变（原始定义） |
| left | `(r,c) → (r, -c)` |
| up | `(r,c) → (-c, r)` |
| down | `(r,c) → (c, r)` |
