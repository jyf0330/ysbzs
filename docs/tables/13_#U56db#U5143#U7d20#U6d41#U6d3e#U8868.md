## B01 · 火伤害（火流派）

| 项目 | 内容 |
|------|------|
| **核心机制** | 叠火层数 → 引爆结算伤害。三角数公式：`N层伤害 = N×(N+1)/2`，克制风(×2) |
| **核心英雄** | 火苗灵(前排叠火)、火种灵(空格引爆增伤)、爆爆灵(死亡铺火)、烈焰使(范围叠火) |
| **辅助英雄** | 余烬灵(持续输出)、熔火灵(高伤害)、龙焰灵(终极输出) |
| **关键标签** | 火、输出、叠层、引爆、前排、范围 |
| **关键决策** | 叠层 vs 引爆时机的把握；空格控制 vs 直接叠怪物格 |
| **爽点** | 「叠满 6 层 + 克制 + 空格加成 → 一发入魂 30+ 伤害」 |
| **强度曲线** | 前期★ 中期★★★ 后期★★★ |
| **当前完整度** | ✅ 火叠层、引爆、克制、空格加成全部实装 |
| **风险** | 数值膨胀（叠层太快会导致怪物完全无压力） |

### 决策路径图

```
Day1-3: 火苗灵 + 火种灵 → 学叠火+引爆
  ↓
Day3-5: 找余烬灵/烈焰使 → 提升输出效率
  ↓
Day5-7: 合成黄金火英雄 → 爆爆灵铺火链
  ↓
Day7-10: 熔火灵/龙焰灵 → 终极火阵容
```

## B02 · 水召唤（水流派）

| 项目 | 内容 |
|------|------|
| **核心机制** | 用水元素格召唤水灵，配合被动增益形成召唤海。层数越高召出水灵 HP 越高 |
| **核心英雄** | 召芽灵(召唤核心)、泉泉灵(治疗引擎)、绒语灵(召唤增益)、分分灵(召唤分裂) |
| **辅助英雄** | 泡泡灵(前排防御)、指挥芽(召唤指挥) |
| **关键标签** | 水、召唤、治疗、增益、分裂、人海 |
| **关键决策** | 什么时机召唤 vs 先叠元素；治疗资源分配 |
| **爽点** | 「3 个水灵在场 + 绒语灵 buff + 泉泉灵治疗 → 怪根本打不死」 |
| **强度曲线** | 前期★ 中期★★★ 后期★★★★ |
| **当前完整度** | ✅ 召唤引擎已实装（召芽/泉泉/绒语/分分/爆爆/指挥芽 6 英雄被动全部实现） |
| **风险** | 召唤海一旦成型对其他流派形成碾压，需要控制召唤数上限 |

### 决策路径图

```
Day1-3: 召芽灵(核心) + 泡泡灵(前排)
  ↓
Day3-5: 召芽灵合成白银+ → 泉泉灵(治疗)加入
  ↓
Day5-7: 绒语灵(召唤buff) + 分分灵(分裂) → 召唤海雏形
  ↓
Day7-10: 指挥芽(终极) → 满场水灵碾压
```

## 流派克制关系

```
火伤害(B01) 强势 → 风牵制(B03) ← 怪物多风时火爽
   ↑                            ↓
土召唤(B10)                 火土生存(B08)
   ↓                            ↑
水召唤(B02) 强势 → 风火速攻(B09) ← 召唤太多时火可撕
```

当前阶段流派设计原则：
1. **不强制绑定**：玩家可以混合不同流派元素（如 2 个火 + 1 个水）
2. **不惩罚混合**：混搭有混搭的强度，不要求纯流派
3. **流派红利可叠加**：同流派英雄越多，协同效果越强
