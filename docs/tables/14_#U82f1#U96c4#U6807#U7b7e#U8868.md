# 14 · 英雄标签表（v1.0）

> 用途：定义标签体系，用于商店筛选、构筑推荐、流派标识
> 数量：35 个标签（4 元素 + 10 定位 + 6 技能 + 8 功能 + 7 被动）

---

## 标签体系总览

### 一、元素标签（4）

| 标签 | 含义 | 使用方式 |
|------|------|----------|
| 火 | 英雄元素 = fire | 属性筛选 |
| 水 | 英雄元素 = water | 属性筛选 |
| 风 | 英雄元素 = wind | 属性筛选 |
| 土 | 英雄元素 = earth | 属性筛选 |

### 二、定位标签（10）

| 标签 | 含义 | 判断标准 |
|------|------|----------|
| 前排 | 近战位，HP 较高 | `levels[1].hp ≥ 22` |
| 后排 | 远程位，HP 较低 | `levels[1].hp ≤ 20` |
| 输出 | 以伤害为主要贡献 | 多伤害型槽 |
| 防御 | 以生存/减伤为主要贡献 | 有减伤/阻挡被动 |
| 单点 | 行动槽形状为 1-2 格 | 主要用 sn1/sn2/sn4 |
| 范围 | 行动槽覆盖 3+ 格 | 主要用 sn5+ |
| 远程 | 命中格在 2+ 格外 | 形状偏长 |
| 近战 | 命中格在 1-2 格内 | 形状偏短 |
| 机动 | 可灵活移位/侧击 | 槽位有多方向 |
| 混合 | 多元素/多定位 | tags > 3 或元素混合 |

### 三、技能标签（6）

| 标签 | 含义 | 对应机制 |
|------|------|----------|
| 召唤 | 有召唤技能 | `skill:'summonFromCell'` |
| 治疗 | 有治疗技能 | `skill:'healSummons'` |
| 铺火 | 死亡/被动在场上留火 | `passive.onSummonDeath` |
| 引爆 | 有十字爆炸被动 | `passive.crossExplosion` |
| 分裂 | 有分裂被动 | `passive.splitSproutSummon` |
| 增益 | 有范围 buff 被动 | `passive.buffAllSummons` |

### 四、功能标签（8）

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| 核心 | 流派核心英雄 | 构筑必须 |
| 引擎 | 流派发动机，需要保护 | 商店给高优先级 |
| 被动 | 被动效果英雄 | 站场就给收益 |
| 灵活 | 多元素槽、可切换定位 | 构筑万金油 |
| 清散怪 | 适合清小怪的技能 | 对 swarm/fast 有效 |
| 侧击 | 从侧面/斜方攻击 | 槽位有向左/其他方向 |
| 十字 | 有十字形状槽 | sn12 |
| 终极 | T4 最终英雄 | 构筑终局 |

### 五、被动标签（7）

| 标签 | 含义 | 代码被动类型 |
|------|------|-------------|
| 城堡减伤 | 减少城堡受伤 | `castleReduce` |
| 引信增伤 | 空格引爆加成 | `spaceExplosionBonus` |
| 克制增伤 | 克制元素额外伤害 | `advHitBonus` |
| 召唤增益 | 全体召唤物攻HP加成 | `buffAllSummons` |
| 治疗引擎 | 治疗量加成 | `healAmpBonus` |
| 死亡铺火 | 单位死亡时铺火 | `onSummonDeath` |
| 十字爆炸 | 引爆范围变十字 | `crossExplosion` |

## 商店筛选优先级

| 筛选入口 | 建议使用标签 | 对应的 UI 控件 |
|---------|-------------|---------------|
| 按元素 | 火/水/风/土 | 四个元素图标按钮 |
| 按位置 | 前排/后排 | 切换行 |
| 按功能 | 输出/防御/召唤/治疗/增益/控制 | 多选标签 |
| 按稀有度 | T1/T2/T3/T4 | 品级筛选 |
| 搜索 | 所有标签 | 输入框匹配 |
| 流派推荐 | 流派名 + 核心 | 推荐构筑列表 |

> **注**：标签系统当前仅在 data.js 中定义，尚未接入 UI 筛选。实施商店标签筛选是 S2 优先级任务。
