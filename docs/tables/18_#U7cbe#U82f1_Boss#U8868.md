### 2.2 精英能力详细

#### 炎灵精英（elite_fire）
```
特性：炼火身躯
能力：每 2 回合（onEveryNthRound, n=2）在自己格铺 1 层火
战术：靠近它的英雄会被叠火 → 需要轮换或快速击杀
克制：水英雄克制火，打它 ×2 伤害
```

#### 风灵精英（elite_wind）
```
特性：飘忽不定
能力：受击时（onTakeDamage）后跳 1 格（优先远离攻击者方向）
战术：远程攻击或范围覆盖可压制
克制：火英雄克制风（但需要命中，后跳可能让火槽空放）
```

#### 岩灵精英（elite_earth）
```
特性：硬甲外壳
能力：每回合首次受击减免 1 伤害（onBeforeTakeDamage, reduction=1, oncePerRound）
战术：用多段伤或召唤物磨死
克制：风英雄克制土 ×2
```

#### 召唤精英（elite_summon）
```
特性：召唤援军
能力：每 3 回合（onEveryNthRound, n=3）在自身后方相邻空格召 1 只 normal
战术：尽快击杀本体，否则小怪越打越多
克制：范围伤害/清场能力强
```

### 3.2 现有 Boss 能力定稿

#### 炎甲兽（boss5）- 铁壁
```
特性：高温护甲（已定义 iron_wall，pending）
能力：当本回合受到的累计伤害 ≥5 时，第 2 次及之后的伤害 -2
触发：onBeforeTakeDamage

战术：高频低伤可以绕过减伤，用召唤物或火焰小刀打
```

#### 熔岩核心（boss8）- 熔岩涌流
```
特性：路过留火（已定义 lava_surge，pending）

状态：🟡 pending → 本次定稿
战术：不要让它走到我方阵地铁！火会铺到己方格子
```

```
状态：🟡 pending → 本次定稿
```

### 3.3 新增 Boss 能力

#### 冰晶守卫（boss_crystal）
```
特性：冻结光环（freeze_aura）
能力：每小回合结束时，自身周围 3×3 范围内所有元素格移除 1 层
触发：onRoundEnd
战术：用叠层打它需要连续叠一样的元素，混搭会被洗掉
```

#### 虚空行者（boss_aether）
```
特性：相位变换（phase_shift）
能力：每 3 回合瞬移到棋盘随机位置，留下 2 层虚空元素（= 土）
触发：onEveryNthRound, n=3
战术：不能让英雄锁定一个位置打必须做好机动准备
```

#### 大地泰坦（boss_titan）
```
特性：大地震击（ground_pound）
能力：每 2 回合对全体英雄/召唤物立即造成 2 点伤害（无视位置）
触发：onEveryNthRound, n=2
战术：英雄 HP 必须足够高，召唤物可能被直接震死
```
