## 一、纯流派路线（4）

### R01 · 火叠爆流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 火苗灵(前排叠火)、火种灵(空格引爆加成)、余烬灵(输出) |
| 辅助英雄 | 烈焰使(范围火)、爆爆灵(死亡铺火)、熔火灵(高输出) |
| 终局英雄 | 龙焰灵 / 火魔(十字爆炸) |
| 关键标签 | 火/输出/叠层/引爆/前排/范围 |
| 关键决策 | 叠层 vs 引爆时机；控制格子防止怪物绕路 |
| 启动难度 | ★☆☆ 新手友好 |
| 强度曲线 | 前期★★ 中期★★★★ 后期★★★★ |
| 弱点 | 怕怪物分散、怕元素被克制(水克火) |

**购买路线**：
```
D1: 火苗灵(必买) + 火种灵(如有)
D2: 余烬灵 → 积累火英雄
D3-4: 找烈焰使(范围火) → 白银合成
D5: 如果有钱 → 熔火灵 替换低级火
D6-8: 滚火英雄雪球 → 看到爆爆灵可买
D9-10: 龙焰灵/火魔 → 钻石终局
```

### R02 · 水召唤流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 召芽灵(召唤核心)、泉泉灵(治疗引擎)、绒语灵(召唤增益) |
| 辅助英雄 | 分分灵(分裂)、泡泡灵(前排)、指挥芽(召更多) |
| 终局英雄 | 元芽灵(终极召唤) |
| 关键标签 | 水/召唤/核心/治疗/增益/分裂 |
| 关键决策 | 召唤时机 vs 叠元素；保护召唤物不被清 |
| 启动难度 | ★★☆ 需要组合配合 |
| 强度曲线 | 前期★(缺核心召唤) 中期★★★ 后期★★★★★ |
| 弱点 | 怕 AOE 精英、召芽灵死亡后体系崩 |

**购买路线**：
```
D1: 泡泡灵(前排苟) + 水滴灵(过渡)
D2-3: 必买召芽灵(流派灵魂) → 合成白银
D4: 泉泉灵(治疗) + 绒语灵(增益)
D5-6: 分分灵(分裂) → 召唤海形成
D7-8: 指挥芽 → 召唤效率翻倍
D9-10: 元芽灵 → 终极召唤
```

## 二、混合流派路线（8）

### R05 · 火土生存流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 岩岩灵(前排土墙) + 火苗灵(后排火输出) |
| 辅助英雄 | 磐石灵、余烬灵、烈焰使 |
| 关键标签 | 土/前排/防御/火/输出/后排/生存 |
| 强度曲线 | 前期★★★ 中期★★★★ 后期★★★ |
| 适用玩家 | 稳健型、不喜欢惊险 |

**运营要点**：
- 岩岩灵前排堵路 → 土格挡住 2 路
- 火苗灵/烈焰使后排放心叠火
- 不怕怪物冲到后排

### R06 · 风火速攻流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 十字使(风叠层) + 火苗灵(火输出) |
| 辅助英雄 | 风风灵(克制增伤)、余烬灵、爆爆灵 |
| 关键标签 | 风/范围/火/输出/快攻/叠层 |
| 强度曲线 | 前期★★★ 中期★★★★ 后期★★ |
| 适用玩家 | 喜欢快节奏、不喜欢拖后期 |

**运营要点**：
- 十字使大范围叠风/火
- 火苗灵打高伤引爆
- 目标：D5-6 之前解决战斗

### R07 · 风水控制流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 十字使(风控场) + 召芽灵(水召唤) |
| 辅助英雄 | 风风灵、泉泉灵、绒语灵 |
| 关键标签 | 风/控制/水/召唤/混合 |
| 强度曲线 | 前期★★ 中期★★★ 后期★★★★ |

**运营要点**：
- 风控怪物走位 → 推进水格/火格
- 召唤物补伤害和阻挡
- 双元素操作：需要同时控风和铺水

### R08 · 水火爆发送

| 项目 | 内容 |
|------|------|
| 核心英雄 | 召芽灵(水召唤) + 爆爆灵(火铺火) |
| 辅助英雄 | 火种灵、泉泉灵、余烬灵 |
| 关键标签 | 水/召唤/火/铺火/引爆/爆发 |
| 强度曲线 | 前期★★ 中期★★★ 后期★★★ |

**运营要点**：
- 召芽灵铺水格 → 召唤水灵（火克水会引爆）
- 利用水火互克产生大量引爆伤害
- 爆爆灵在召唤物死亡时铺火 → 形成火海

### R09 · 三元素均衡流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 均衡灵(三元素) |
| 辅助英雄 | 火苗灵、十字使、滴滴灵 |
| 关键标签 | 混合/灵活/均衡 |
| 强度曲线 | 前期★★ 中期★★★ 后期★★★ |
| 适用玩家 | 老手、喜欢灵活应变 |

**运营要点**：
- 均衡灵一个人可打 3 种元素
- 根据场上元素形势切换
- 上限依赖玩家操作

### R10 · 城堡守护流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 泡泡灵(城堡减伤) + 岩岩灵(前排) |
| 辅助英雄 | 磐石灵、泉泉灵、召芽灵 |
| 关键标签 | 防御/减伤/生存/前排/城堡 |
| 强度曲线 | 前期★★★★ 中期★★★★ 后期★★★ |

**运营要点**：
- 泡泡灵城堡减伤(1-2点) + 岩岩灵前排阻挡
- 城堡几乎不掉血
- 缺点是输出低、战斗时间长

### R11 · 火魔终结流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 火魔(钻石/十字爆炸) |
| 辅助英雄 | 爆爆灵、火种灵、熔火灵 |
| 关键标签 | 火/爆炸/终极/十字/钻石 |
| 强度曲线 | 前期★(没成型前弱) 中期★ 后期★★★★★ |
| 适用玩家 | 喜欢憋大招、后期爆发的 |

**运营要点**：
- 前 7 天全力攒钱、苟且偷生
- D7 看到火魔时全力购买+合成
- 火魔十字爆炸被动 + 最高叠层 = 清屏
- 前 7 天只要城堡不爆就行

### R12 · 召唤海淹流

| 项目 | 内容 |
|------|------|
| 核心英雄 | 召芽灵 + 分分灵 + 绒语灵 + 指挥芽 |
| 辅助英雄 | 元芽灵、泉泉灵 |
| 关键标签 | 召唤/分裂/增益/人海/终极 |
| 强度曲线 | 前期★ 中期★★★ 后期★★★★★★ |
| 适用玩家 | 喜欢人海战术、滚雪球 |

**运营要点**：
- 全场铺满水灵
- 分分灵让召唤物越打越多
- 绒语灵让召唤物越打越强
- 泉泉灵让召唤物死不了
- ⚠️ 这是当前版本最可能超模的流派
