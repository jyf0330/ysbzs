## 四、难度解锁

| 难度 | 解锁条件 |
|:----:|---------|
| 普通 | 默认解锁 |
| 困难 | 通关 1 次普通 |
| 挑战 | 通关 3 次普通 或 1 次困难 |
| 自定义 | 通关挑战后解锁倍率滑块（未来功能） |

## 五、实现建议

难度参数只需在 `game.js` 初始化时加载一个倍率对象，所有怪物生成和结算引用该对象。
代码改动量：约 2 小时。
