# 25 · 战斗日志模板表（v1.0）

> 用途：定义所有战斗日志的模板和格式
> 数量：100 条模板（当前定义 50 条核心模板）
> 注意：日志主要面向开发调试；玩家可折叠，默认收起

---

## 一、日志格式规范

```
[回合][类别] 内容...
```

| 字段 | 值 |
|------|----|
| 回合 | `[D3-M2-R3]` = Day3 上午 回合3 |
| 类别 | `[行动]` `[元素]` `[伤害]` `[怪物]` `[召唤]` `[商店]` `[系统]` |
| 颜色 | 火=红、水=蓝、风=绿、土=棕、伤害=红、系统=灰 |

## 十、实现建议

1. 日志保存在 `window.gameState.battleLog = []`
2. 每次新增用 `unshift` 倒序（最新在上）
3. UI 渲染时按类别标色
4. 最大保留 200 条，超过丢弃最旧
5. 开发者模式显示 `[DEBUG]` 级别
