# 26 · 新手引导镜头/高亮表（v1.0）

> 用途：补充 09_新手教程表.md 的视觉引导定义
> 当前教程表已有 60 步文字引导，本表定义每一步的高亮/镜头/交互
> 数量：60 步关联

---

## 一、引导视觉元素

| 元素 | 用途 | 样式 |
|------|------|------|
| 🔲 高亮框 | 指向可交互元素 | 金色发光边框 + 脉动动画 |
| 👆 点击提示 | 指示玩家点击 | 手指图标 + 位置箭头 |
| 📖 文字弹窗 | 显示教学文案 | 羊皮纸风格文字框 |
| 🔄 区域圈选 | 圈出一片操作区域 | 半透明遮罩 + 开放光圈 |
| 🎯 目标标记 | 指向目标格子 | 闪烁圆圈 |
| ⏸️ 暂停覆盖 | 阻止玩家提前操作 | 半透明蒙层 |

## 二、核心引导步骤的视觉设计

### D1 · 第一回合（最重要）

```
步骤 T001: "这是你的英雄"
视觉: 🔲 高亮框 → 火苗灵英雄栏
       📖 文字 → 英雄栏上方 "点击英雄查看行动槽"
操作: 等待玩家点击火苗灵
通过: 火苗灵被选中
```

```
步骤 T002: "选择行动槽"
视觉: 🔲 高亮框 → 火苗灵的 3 个行动槽
       🔄 区域圈选 → 整个英雄栏变暗，仅行动槽点亮
       📖 文字 → "行动槽决定叠元素的位置和形状"
操作: 等待玩家点击任意行动槽
通过: 行动槽被选中
```

```
步骤 T003: "查看命中格预览"
视觉: 🔲 高亮框 → 棋盘上被命中的格子
       🎯 目标标记 → 每个命中格有闪烁边框
       📖 文字 → "蓝色高亮的格子会被叠火层"
通过: 等待 2 秒（自动通过）
```

```
步骤 T004: "确认行动"
视觉: 🔲 高亮框 → 棋盘下方的"确认"按钮
       📖 文字 → "点击确认完成本英雄的行动"
操作: 点击确认按钮
通过: 按钮被点击
```

### D1 · 商店第一次进入

```
步骤 T101: "欢迎来到商店"
视觉: 🔲 高亮框 → 商店面板整体
       📖 文字 → "在这里购买英雄，同名英雄合成升阶"
通过: 等待 2 秒
```

```
步骤 T102: "购买英雄"
视觉: 🔲 高亮框 → 第一个商品（火苗灵）
       📖 文字 → "点击购买火苗灵"
操作: 点击购买
通过: 购买成功
```

```
步骤 T103: "合成英雄"
视觉: 🔲 高亮框 → 如果抽到同名→合成按钮亮起
       📖 文字 → "同名自动合成！品级提升"
通过: 等待 2 秒
```
