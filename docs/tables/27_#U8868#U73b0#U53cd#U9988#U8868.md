## 六、技术方案

| 反馈类型 | 实现方式 | 复杂度 |
|---------|---------|:------:|
| CSS 动画（高亮/闪烁/脉动） | CSS keyframes + class 切换 | ★ |
| DOM 飘字 | 创建临时 div + 动画结束移除 | ★★ |
| Canvas 粒子 | Canvas layer overlay | ★★★ |
| 屏幕震动 | body transform: translate 抖动 | ★ |
| 面板滑动 | CSS transition | ★ |
| 融合/合成 | DOM 层级叠加 + opacity transition | ★★ |
