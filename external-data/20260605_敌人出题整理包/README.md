## 使用顺序

1. 先看 `2_怪物波次表_Day1-Day5_20260605.xlsx`
2. 再看 `1_宠物总表_127_数值定位版_20260605.xlsx`
3. 需要拆机制时看 `1_机制词条库_魔塔尖塔可计算版_20260605.xlsx`

5. 需要查公式或边界时看 `planner_table_rules_20260605.yaml`、`planner_table_notes_20260605.md`
6. 需要接运行时事件数据时看 `ysbzs_events_20260605.yaml` 和 `ysbzs_events_接入说明_20260605.md`
