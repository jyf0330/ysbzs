## 你日常怎么改

1. 改宠物基础信息：去 `01_pets`。
2. 改最终战斗数值：去 `02_monster_templates`。
3. 改出怪节奏：去 `03_monster_waves`。
4. 改机制：去 `04_机制词条库_60条`。
5. 改事件：去 `05_事件主表_好读版`。
6. 改商店/奖励池：去 `06_商店奖励池_好读版`。
7. 改遗物/祝福：去 `07_遗物祝福_好读版`。
8. 改形状/行动槽摘要：去 `08_形状行动槽_好读版`。
9. 查断链：去 `09_跨表校验`，缺失数量必须为 0。
