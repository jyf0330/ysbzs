## Codex Goal Charter

### 目标

按 `assets/references/doodle_tactics_66.png`（Doodle Tactics 风格参考图）对 `index.html` 进行全面视觉风格重制，使整体界面达到"暖色纸面 + 手绘涂鸦卡牌"美感，同时保持游戏核心逻辑零改动、`node test.js` 通过率不低于基线（当前 405/422）。

### 参考图解析（doodle_tactics_66.png）

| 区域 | 特征 |
|---|---|
| 背景 | 暖沙色/奶油纸面，约 `#e8d5b0`，无硬分区线 |

| 中央棋盘 | 8×8，极淡虚线格，坐标轴 A-H（列）/ 1-8（行），棋子下方显示小 HP 条 |

| 整体色调 | 低饱和暖色，无深色暗角，纸面质感 |

### 范围

**必须改动：**
- CSS 变量 / 色板：向 66.png 色调对齐（更暖、更浅、更柔和）
- 背景：`#game-shell` 换成纯暖奶油纸面，去掉深色网格线
- 左侧栏：从 Debug Dock 改为英雄卡组（`#hero-list` 区域）— 含圆形数字徽章、角色 emoji、HP 条、DEF 显示
- 右侧栏：符文行动卡区域样式改为敌方卡列表 + 消息盒风格（结构可保留，视觉重写）
- 棋盘：格子边框改为淡虚线，棋盘底色更浅，坐标轴字体圆润
- 顶栏：改为居中圆角药丸式回合计数器 + 右侧圆形功能按钮
- 底部：增加技能卡栏或重设底部日志为技能卡风格
- 卡片通用样式：圆角 + 虚线描边 + 奶油底色（`.doodle-card` 通用类）
- 棋子牌面：英雄/怪物标记改为更大、更可爱的圆形徽章风格

**禁止改动：**
- 任何游戏逻辑、战斗计算、数值、状态机
- `node test.js` 中的任何测试用例
- 存档/回放/录制机制
- HTML 结构 ID（`#board`, `#asl`, `#debug-dock`, `#top-bar`, `#log` 等）可重用，但不得删除有逻辑绑定的 ID

### 分阶段执行

#### Phase 1：CSS 变量 + 背景 + 棋盘（最小可见效果）

- 更新 `:root` CSS 变量色板向 66.png 对齐
- `#game-shell` 背景换为纯奶油纸面（去除深色格子纹、暗角）
- `#board` 格子改为虚线边框、浅底色
- 棋盘背景更浅

#### Phase 2：顶栏 + 卡片通用样式

- 顶栏重设为居中圆角药丸计数器
- 定义 `.doodle-card` 通用类：圆角、虚线描边、奶油底
- 数字/字母徽章 `.badge-num` / `.badge-letter`：圆形、元素色

#### Phase 3：左侧英雄面板 + 右侧敌方面板

- 左侧：英雄列表卡组，角色 emoji + HP 条 + DEF
- 右侧：敌方卡列表 + 消息盒（复用现有行动槽 HTML，重写 CSS + JS 渲染）

#### Phase 4：底部技能卡栏

- 底部改为横向技能卡列表（6 格），大图标 + 名字 + 费用水滴

#### Phase 5：棋子牌面 + 细节打磨

- 棋子圆形徽章、HP 条精化
- 全局字体、间距、阴影微调
- 浮动伤害数字样式保留/优化
